## Installation

```sh
git clone https://github.com/andre347/tableau-embedding-api-v3.git

cd tableau-embedding-api-v3/

# run this if you use yarn
yarn
# run this if you use npm
npm install
# start the application (yarn)
yarn run dev
# start the application (npm)
npm run start
```
